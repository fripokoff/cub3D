## Compile and Run

### Usage `./Cub3D [textures] [map_size]`

### For compile program and mlx
```bash
make
```

## Norm
```bash
./make norm
```

### Run the program
```bash
./cub3D
```

### Choose map
- Cardinal map = 0
- minecraft map = 1
```bash
./cub3D 0
```

### Choose size of map
- Very small = 0
- Small = 1
- Big = 2
```bash
./cub3D 0 1
```
